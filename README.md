# GOD SYSTEM Lua ![](https://img.shields.io/badge/2.7.1-7393B3)
![](https://img.shields.io/badge/Game_version-1.68-yellow) ![](https://img.shields.io/badge/Game_build-3179-yellow)

Lua for Midnight GTA:O.

# Installation
Move files from archive to `\lua` folder.

# Functions
### Businesses
- Office
  - TP to office
  - Instant cargo sell
  - Crates loop
- Bunker
  - TP to bunker
  - Get supplies
- Acid lab
  - Get supplies
- Hangar
  - Instant cargo sell
  - Cargo loop
- Clubhouse
  - Get supplies
  - Get all supplies
### Casino
- Chips
  - Chips purchase limit
  - Reset chips purchase cooldown
- Lucky wheel
  - Give drop from lucky wheel
- Blackjack
  - Trick The Dealer
- Poker
  - Give straight flush
  - Trick The Dealer
### Spawner
- Spawn by name
- Spawn from 'mods' folder
- Spawned vehicles
  - Delete selected vehicle
  - Delete all
### Misc
- Unlocks
  - Unlock all removed vehicles
  - Unlock LS Car Meet podium prize
  - TP to gun van
- Loops
  - Nigh Club loop ($300k/2s)
  - Ships loop (5k/3s)
  - Death loop ($50k/1s)
