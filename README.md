# GOD SYSTEM Lua

![](https://img.shields.io/badge/Game_version-1.72-blue) ![](https://img.shields.io/badge/Game_build-3751-blue) </br>
Lua for [MIDNIGHT GTA:O](https://midnight.im/).
To-Do list and [`beta branch`](https://github.com/Legionidk/GOD-SYSTEM-Lua/tree/beta) changelogs can be found [here](https://any.coop/A87gcpZJWCrFRfzP88idEK4pkm7BzQ3cZvN69A6bDxN1378w/god-system).

# Installation

Move files (`godSystem.lua` and `godSystem folder`) from release archive to `<midnight folder>\lua`.

# Current functions

## Businesses

<details>
<summary>Bunker</summary>

```
- Stock, research and supplies info
    - Bunker stock display
    - Bunker research display
    - Bunker supplies display
    - Teleport to your bunker

- Stock price
    - Multiply stock price
    - Reset stock price

- Stock production
    - Stock production loop
    - Trigger stock production

- Sell stock
    - Instant sell

- Supplies
    - Get supplies
```

</details>

<details>
<summary>Office</summary>

```
- Crate price
    - Set new crate price
    - Reset crate price

- Sell crates

- Cooldowns

    - Remove sell mission cooldown
```

</details>
